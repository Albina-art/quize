## Quiz Trainer (Next.js + PostgreSQL)

Приложение для сохранения вопросов/ответов по темам и случайной выдачи вопросов.

### Что умеет

- Добавлять тему, вопрос, подсказку и ответ в PostgreSQL.
- Получать случайный вопрос из базы.
- Показывать и скрывать подсказку и ответ для самопроверки.

Готовый набор из пяти вопросов по теме **CORS** заливается командой `npm run db:seed` **только после** применения схемы к БД (шаг 2 ниже). Повторный сид не дублирует карточки с тем же текстом вопроса.

### Запуск

1. Поднимите PostgreSQL:

```bash
docker compose up -d
```

2. Создайте таблицы в базе (без этого `db:seed` выдаст ошибку «table does not exist»):

```bash
npx prisma migrate dev
```

Без истории миграций можно вместо этого: `npx prisma db push`.

3. Загрузите вопросы по CORS:

```bash
npm run db:seed
```

4. Запустите приложение:

```bash
npm run dev
```

Откройте `http://localhost:3000`.
